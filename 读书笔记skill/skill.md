# 读书笔记skill

这是一个用于生成学术读书笔记的claude code skill，适用于经济学、政治学等社会科学专业的读书笔记作业。

## 使用场景

当用户需要完成读书笔记作业时，调用此skill。根据用户要求，生成符合学术规范的读书笔记，可导出为Word文档。

## 输入

- 用户提供的书籍信息（书名、作者、或让skill推荐）
- 是否有特殊的框架要求
- 是否需要按照《经济研究》等权威期刊格式排版

## 核心原则

### 1. 语言润色要求
- 使用段落式排布而非分点排布，使行文流畅如可供发表的文章
- 避免碎片化的要点罗列，每个段落应有完整的论证逻辑
- 句子之间应有自然的衔接和过渡
- 学术写作应严谨但不生硬，表达清晰但不浅白

### 2. 学术性要求（参考聂辉华逻辑）
- 每个问题的回答都应有学术深度
- 论证观点时：不仅概括表面结论，还要分析理论机制和逻辑链条
- 核心观点时：提炼出简洁有力的学术命题，并解释其理论含义
- 例证阐述时：不仅描述案例，还要分析案例背后的因果机制
- 实证方法时：说明方法选择的合理性及其局限性

### 3. 格式要求（《经济研究》期刊风格）
- 页面设置：A4纸，上下2.54cm，左右3cm/2.54cm
- 正文字体：宋体小四（12pt），行距22pt
- 一级标题：黑体三号（14pt），居中
- 二级标题：宋体小四（12pt），加粗
- 参考文献：宋体五号（10.5pt），悬挂缩进
- 中文标点混排时注意全角/半角规范

## 输出框架

### 标准框架

```
一、引言（可选）
   - 书籍的学术地位和重要性
   - 本笔记的写作目的和结构安排

二、书籍概况
   2.1 作者简介
       - 学术背景、职称、主要研究领域
       - 重要著作和学术荣誉（如诺贝尔奖等）
       - 跨学科研究特点（如果有）

   2.2 书籍简介
       - 出版信息、主旨和核心论点
       - 学术影响和政策意义

   2.3 书籍框架
       - 全书结构和章节安排（段落式描述，非表格）

三、书籍内容
   3.1 本书的论证观点（三个层面的概括）
   3.2 本书的核心观点（理论命题式表述）
   3.3 作者的例证分析（结合理论机制分析案例）
   3.4 本书的实证研究方法（说明方法论的逻辑）
   3.5 章节内容详述（每个部分的要点概括，段落式）

四、思考与启示
   4.1 与专业的联系（理论、方法论、政策三个维度）
   4.2 心得体会（制度概念、制度形成机制、历史偶然性等）
   4.3 简要评价（优缺点、学术价值、局限性）

参考文献
```

## 输出格式

根据用户需求，可选择：
1. **Markdown格式**：适合进一步编辑和转换为其他格式
2. **Word文档**：按照《经济研究》期刊格式排版，适合直接提交

## Skill文件结构

```
读书笔记skill/
├── skill.md                    # 本说明文件
├── framework.md               # 详细框架模板（含写作指南）
├── prompts/
│   ├── book-recommendation.md  # 推荐书籍场景
│   └── writing-guidance.md     # 写作指导提示词
└── examples/
    ├── why-nations-fail.md     # 示例（完善前）
    └── why-nations-fail-polished.md  # 示例（完善后）
```

## 使用方法

1. 用户提供书籍信息
2. 确定是否需要按特定期刊格式排版
3. 生成读书笔记草稿
4. 根据用户反馈进行润色和完善
5. 如需Word文档，执行格式转换
